# METT517-FocusQuest

**An enhanced version of [focus-sprint-coach]** - Pomodoro-style focus timer with real-time energy tracking, gamification mechanics, and qualitative check-ins.

> ⚠️ **Note**: This project is an improved iteration based on the original focus-sprint-coach. It is not my original creation but a derivative work that adds gamification features and enhanced user experience.

## 🎯 About This Project

FocusQuest is inspired by and built upon **focus-sprint-coach**, adding:
- 🎮 Gamification elements (levels, badges, streaks)
- 💎 Crystal collection and skill unlock system
- 📊 Enhanced energy pattern insights
- 🏆 Achievement system for motivation

The core timer and feedback mechanism remain faithful to the original project's philosophy of combining Pomodoro technique with emotional and energy check-ins.

## 📖 한국어 설명

**[focus-sprint-coach]의 개선 버전** - 실시간 에너지 추적, 게이미피케이션, 감정 체크인 기능이 있는 뽀모도로 타이머입니다.

> ⚠️ **알림**: 본 프로젝트는 원본 focus-sprint-coach를 기반으로 개선한 파생 작업물입니다. 제 고유한 창작물이 아니며, 게이미피케이션 기능과 향상된 사용자 경험을 추가한 버전입니다.

## 🎯 프로젝트 소개

FocusQuest는 **focus-sprint-coach**에서 영감을 받아 제작되었으며, 다음 기능을 추가했습니다:
- 🎮 게이미피케이션 요소 (레벨, 배지, 연속 달성)
- 💎 크리스탈 수집 및 스킬 해금 시스템
- 📊 강화된 에너지 패턴 인사이트
- 🏆 동기부여를 위한 업적 시스템

뽀모도로 기법과 감정/에너지 체크인을 결합한 원본 프로젝트의 철학을 유지하면서 핵심 타이머와 피드백 메커니즘을 충실히 따르고 있습니다.
